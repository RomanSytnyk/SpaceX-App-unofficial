# SpaceX-App-unofficial
Fan unofficial SpaceX Android App

**The repository is open for suggestions and improvements.**

Libraries:
- Android Support Library
- Moxy (MVP)
- Glide
- Retrofit 2 + Coroutines Adapter
- Dagger 2
